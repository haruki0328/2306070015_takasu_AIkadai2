# BookShelf Pro

書籍を検索し、レビューを投稿・閲覧しながら、自分の読書状況（読みたい／読書中／読了）を管理できる Streamlit 製の Web アプリです。Open Library API から書籍データを取得し、ユーザーごとのレビュー・本棚情報は CSV ファイルで管理します。

## 主な機能

- **ユーザー認証**
  ユーザー名・パスワードによる新規登録／ログイン。パスワードは SHA-256 でハッシュ化して保存し、登録時には「8文字以上・大文字・小文字・数字を含む」ことを要件としてチェックします。

- **書籍検索＋ランダムおすすめ**
  キーワードで Open Library から書籍を検索できます。キーワード未入力時はトップに「今日のおすすめ」としてランダムなジャンルの本を表示し、「🎲 シャッフル」ボタンで何度でも入れ替えられます。検索結果が0件のときは、単語単位で緩めた条件での再検索を自動で試みたうえで、見つからない場合も画面が空にならないようおすすめ本を表示します。

- **レビュー投稿・いいね機能**
  各書籍の詳細ページで5段階評価とコメントを投稿できます。投稿されたレビューはいいね数順に表示され、他ユーザーのレビューに「👍 いいね」を付けられます。

- **ランキング（トレンド）**
  レビュー数の多い書籍（トレンド）と平均評価の高い書籍（トップレーティング）をそれぞれランキング表示します。

- **読書ステータス管理／本棚ダッシュボード**
  書籍ごとに「未登録／読みたい／読書中／読了」のステータスを設定でき、マイページではステータス別の冊数集計と、本棚のようなグリッド表示で自分の本を一覧できます。

- **関連書籍のおすすめ**
  書籍詳細ページでは、同じ著者の他の作品を自動検索して関連書籍として表示します。

##  使用API

- [Open Library API](https://openlibrary.org/developers/api)
  - 書籍検索: `https://openlibrary.org/search.json`
  - ジャンル別書籍取得（ランダムおすすめ用）: `https://openlibrary.org/subjects/{subject}.json`
  - 書影画像: `https://covers.openlibrary.org/b/id/{cover_id}-M.jpg`

## ファイル構成

| ファイル | 役割 |
|---|---|
| `app.py` | Streamlit による画面・UI制御（ログイン、検索、ランキング、マイページ、詳細＆レビュー画面） |
| `logic.py` | Open Library API通信、検索ロジック、認証、レビュー／ステータス管理などのデータ処理 |
| `users.csv` | ユーザー情報（ユーザー名・ハッシュ化パスワード） |
| `reviews.csv` | レビュー情報（評価・コメント・いいね数など） |
| `user_books.csv` | ユーザーごとの読書ステータス（読みたい／読書中／読了） |
| `requirements.txt` | 必要なPythonパッケージ一覧 |
| `improvement.md` | 今後の改善提案（パーソナライズ・レコメンドエンジン構想など） |
| `docs/` | 設計図・構成イメージなどの資料 |
| `.devcontainer/` | GitHub Codespaces / VS Code Dev Containers 用の開発環境設定 |

##  技術構成

```
[ ユーザー（ブラウザ） ]
        │
        ▼
[ Streamlit アプリ (app.py) ]
        │
        ▼
[ ロジック層 (logic.py) ]
   ├── 認証・パスワードハッシュ化
   ├── 書籍検索／ランダムおすすめ（Open Library API 呼び出し）
   ├── レビュー・いいね管理
   └── 読書ステータス管理
        │
        ▼
[ CSVファイル群 ]
   users.csv / reviews.csv / user_books.csv
```

## セットアップ・使い方

### 1. 必要なパッケージのインストール

```bash
pip install -r requirements.txt
```

主な依存パッケージ:

- `streamlit`
- `pandas`
- `requests`

### 2. アプリの起動

```bash
streamlit run app.py
```


### 3. 利用の流れ

1. ブラウザで起動した画面にアクセスします（デフォルトでは `http://localhost:8501`）。
2. 「新規登録」からアカウントを作成し、ログインします。
3. サイドバーから以下のページを切り替えて利用できます。
   - **🔍 検索する**：キーワード検索、またはおすすめ本の閲覧　
   - **📈 トレンド**：レビュー数・評価によるランキング
   - **📚 本棚ダッシュボード**：自分の読書ステータス管理

##  注意事項

- ユーザー情報・レビュー・読書ステータスはすべてローカルのCSVファイルに保存されるため、複数人での同時利用やデプロイ環境によってはデータの取り扱いに注意してください。
- Open Library API の応答状況によっては、検索結果やおすすめ本が取得できない場合があります。
- 日本の書籍のAPIがなかったので海外のAPIを使っています。英語の本ばかりで日本語の検索が上手く機能しないです。基本的に英語で検索してください。
